# Aplicações Linux

## 🌐 Web e Comunicação
Ferramentas para navegação, gestão de e-mails e comunicação em tempo real.

* **Navegadores:**
    * **Firefox:** Navegador padrão com foco em privacidade e código aberto.
    * **Google Chrome / Chromium:** Alta compatibilidade e performance.
    * **Brave:** Focado em privacidade e bloqueio nativo de anúncios.
* **Clientes de E-mail:**
    * **Thunderbird:** Completo, com suporte a extensões e calendários.
    * **Evolution:** Ideal para integração com ambientes corporativos (GNOME).

## 📂 Produtividade e Office
Suítes completas para trabalho administrativo e acadêmico.

* **LibreOffice:** A alternativa de código aberto mais utilizada (Writer, Calc, Impress).
* **OnlyOffice:** Excelente compatibilidade com arquivos `.docx`, `.xlsx` e `.pptx`.

## 💻 Ferramentas de Desenvolvimento
O Linux é amplamente reconhecido como a melhor plataforma para programadores.

* **Editores e IDEs:**
    * **VS Code:** O editor mais versátil com vasta biblioteca de extensões.
    * **Vim / Neovim:** Editores de terminal focados em produtividade extrema.
* **Compiladores e Debuggers:**
    * **GCC / G++:** O padrão para desenvolvimento em C e C++.
    * **GDB:** Ferramenta clássica para depuração de código.
* **Versionamento:**
    * **Git:** Essencial para controle de versão e colaboração.

## 🎬 Multimídia (Áudio e Vídeo)
Soluções para reprodução, gravação e edição de mídia.

* **Players:**
    * **VLC Media Player:** Suporta virtualmente qualquer formato de mídia.
    * **MPlayer**
* **Edição e Captura:**
    * **Blender**
    * **Cinelerra**
    * **Audacity:** Editor de áudio robusto para edição e mixagem.

## 🎨 Gráficos e Design
Ferramentas para manipulação de imagem, desenho vetorial e 3D.

* **GIMP:** O principal manipulador de imagens (alternativa ao Photoshop).
* **Inkscape:** Software profissional para ilustração vetorial (SVG).
* **Blender:** Suite completa para modelagem, animação e renderização 3D.

---
> **Nota:** No Linux, essas aplicações podem ser instaladas via gerenciadores de pacotes nativos (`apt`, `dnf`) ou via pacotes universais como **Flatpak** e **Snap**.
