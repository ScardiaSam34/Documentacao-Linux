# Introdução ao Linux 🐧

O **Linux**, diferentemente do que se acredita, não é necessariamente o sistema operacional, mas sim a junção de diversas funcionalidades e programas junto de um *kernel* (núcleo) de código aberto que serve de base para diversos sistemas conhecidos como **Distribuições** (ou *Distros*).


## Arquitetura do Sistema
Para entender o Linux, é preciso visualizar como ele se organiza entre o hardware e o usuário:

1.  **Hardware:** A parte física (CPU, Memória, Disco).
2.  **Kernel:** O coração do sistema. Gerencia o hardware e os recursos.
3.  **Shell:** A interface que interpreta os comandos do usuário.
4.  **Aplicações:** Programas de interface gráfica ou linha de comando.


##  Principais Características
*   **Open Source:** O código é aberto e pode ser modificado por qualquer pessoa.
*   **Multitarefa e Multiusuário:** Projetado para lidar com vários processos e pessoas simultaneamente.
*   **Segurança:** Baseado em permissões rígidas de arquivos.
*   **Estabilidade:** Amplamente utilizado em servidores e supercomputadores.
